高级 Python 后端工程师

职位描述：
我们正在寻找一位经验丰富的 Python 后端工程师，加入我们的核心架构团队。

岗位职责：
1. 负责公司核心业务系统的后端架构设计与开发
2. 参与微服务架构的设计、拆分和治理
3. 编写高质量代码，保证系统的高性能和高可用
4. 指导初中级工程师，进行代码审查

任职要求：
1. 5年以上 Python 后端开发经验
2. 精通 Django 或 FastAPI 框架，有大型项目经验
3. 熟悉微服务架构，有实际微服务拆分或治理经验
4. 熟练使用 Docker、Kubernetes 进行容器化部署
5. 熟悉 MySQL、Redis、消息队列等中间件
6. 本科及以上学历，计算机相关专业
7. 有团队管理或技术 leader 经验优先

加分项：
- 有高并发系统设计经验
- 熟悉 CI/CD 流程
- 开源项目贡献者
